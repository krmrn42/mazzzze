## F-21: Видоискатель — окно прицеливания

При активации использования (игрок нажимает ЛКМ с фотоаппаратом в руке) на экране появляется окно видоискателя. Окно эмулирует взгляд через объектив старинного фотоаппарата, нацеленного прямо перед игроком.

### Общее описание

| Параметр | Значение |
|----------|----------|
| Что отображается | Вид от первого лица (как если бы игровая камера была прикреплена к персонажу на уровне глаз) |
| Направление | Прямо вперёд по направлению взгляда игрока (направление camera-forward) |
| Положение камеры | Уровень глаз персонажа (примерно Y ≈ 1.5 от пола) |
| FOV (угол обзора) | Суженный относительно основной камеры (имитация реального объектива) |
| Режим движения | Игрок **может продолжать движение** (WASD); вид в видоискателе обновляется в реальном времени соответственно перемещению и повороту |
| Управление камерой | Мышь работает для поворота **влево/вправо** (yaw). Поворот **вверх/вниз** (pitch) заблокирован в режиме видоискателя |
| Синхронизация | Видоискатель движется **синхронно с основной камерой** игрока. Основная камера игрока является первичной — видоискатель следует за ней |
| Съёмка | Снимок делается **автоматически** по завершении таймера; ручной спуск (отдельная кнопка «снять сейчас») не предусмотрен |

### Визуальное представление

| Параметр | Значение |
|----------|----------|
| Тип окна | Прямоугольное (стилизованное под окошко видоискателя старинной камеры) |
| Размер | Достаточно крупный, чтобы различать детали лабиринта, но не во весь экран (примерно 40–60% площади экрана) |
| Расположение | По центру экрана, поверх остального UI |
| Фон | Основная сцена позади видоискателя затемняется (полупрозрачная вуаль), чтобы фокусировать внимание на видоискателе |
| Граница окна | Стилизованная рамка «под старину» (тёмное дерево/латунь, виньетка по углам) |
| Поверх окна | Посередине отображается цифра таймера обратного отсчёта (см. [02-timer.md](./02-timer.md)) |

### Стилистические ориентиры

Окно видоискателя должно создавать впечатление «смотрим через старинную камеру»:
- Рамка с текстурой под дерево/латунь
- Виньетка (затемнение по краям)
- Лёгкая сепия или приглушённый цветовой тон внутри видоискателя
- Возможна лёгкая зернистость или плёночный шум (опционально, для атмосферы)
- Возможен лёгкий эффект хроматической аберрации (опционально)

Точный визуальный стиль будет уточнён на этапе реализации.

### Состояние «съёмка невозможна» (Focus Blocked)

Если расстояние от игрока до ближайшей преграды в направлении взгляда меньше минимального фокусного расстояния (см. F-23), использование фотоаппарата блокируется:

| Параметр | Значение |
|----------|----------|
| Когда отображается | При нажатии ЛКМ, если фокусное расстояние нарушено (видоискатель не открывается, таймер не запускается) |
| Визуальная индикация | Видоискатель отображается, но с явным индикатором блокировки — например, перечёркнутый круг или красный символ по центру, поверх размытого или затемнённого изображения |
| Длительность | Кратковременно (~1 секунда), затем видоискатель закрывается. Либо закрывается сразу при отпускании ЛКМ |
| Поведение после | Фотоаппарат остаётся в состоянии Activated. Игрок может отойти или повернуться и попробовать снова |

### Поведение при движении

| Ситуация | Поведение |
|----------|-----------|
| Игрок идёт вперёд/назад (W/S) | Вид в видоискателе сдвигается вперёд/назад соответственно |
| Игрок стрейфится (A/D) | Вид сдвигается влево/вправо |
| Игрок поворачивает мышь влево/вправо | Вид в видоискателе поворачивается (панорамирование). Основная камера игрока также поворачивается синхронно |
| Игрок двигает мышь вверх/вниз | **Заблокировано** в режиме видоискателя. Камера не наклоняется, вид не смещается по вертикали |
| Игрок прыгает (Space) | Вид в видоискателе смещается по вертикали соответственно прыжку |

**Важно:** видоискатель показывает вид от первого лица *в реальном времени* — это не статичный скриншот, а живая трансляция, синхронизированная с основной камерой игрока. Основная камера является первичной: видоискатель следует за ней, а не наоборот.

### Жизненный цикл видоискателя

1. **Проверка фокусного расстояния:** при нажатии ЛКМ проверяется расстояние до преграды (F-23). Если меньше минимального — показывается состояние «съёмка невозможна», видоискатель и таймер не запускаются
2. **Открытие:** фокусное расстояние в норме → видоискатель появляется на экране. Основная сцена затемняется
3. **Работа:** видоискатель отображает вид от первого лица, обновляемый каждый кадр; таймер тикает (3 → 2 → 1)
4. **Закрытие (успешное):** по завершении таймера (переход на 0) видоискатель закрывается автоматически; создаётся фотография; фотоаппарат уничтожается
5. **Закрытие (прерывание):** при нарушении фокусного расстояния во время таймера, либо при выбрасывании/деактивации — видоискатель закрывается, таймер сбрасывается. Фотоаппарат остаётся в состоянии Activated (при нарушении фокуса) или переходит в соответствующее состояние (при выбрасывании/деактивации)

### Поведение UI во время видоискателя

| Элемент UI | Поведение |
|------------|-----------|
| Миникарта | Продолжает отображаться |
| Инвентарь (I) | Открытие **заблокировано** |
| ЛКМ (повторно) | **Игнорируется** |
| Мышь влево/вправо | Поворот камеры (основная + видоискатель синхронно) |
| Мышь вверх/вниз | Заблокировано |
| WASD | Движение игрока |
| Tab | Переключение ориентации миникарты (штатное поведение) |
| Ctrl+Q | Выход из игры (штатное поведение) |
