# Wukong — Параметры по умолчанию (Data)

> **Фича:** REQ-0020-base-monster-wukong · **Тип документа:** требования, грань «данные» (WHAT, без деталей реализации)

Конкретные значения свойств шаблона ([F-39](../04-data.md)) для Wukong. Числа — по умолчанию; окончательно калибруются при реализации.

## Параметры Wukong

### Идентичность и размер

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Идентификатор типа | `wukong` | Ключ типа монстра |
| Звериный тип | волк | Визуальная/тематическая роль |
| Вариант размера | большой/обычный | Мячом сбивается с толку, но не убивается |
| Целевой габарит модели | **2.8 wu** (по длине) | Масштаб `art/black_myth_wukong.glb` — модель низкая/длинная, вписана в коридор по горизонтали (см. design) |

### Восприятие ([F-40](../02-perception-logic.md))

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Дальность зрения | **18 wu** (≈5 клеток) | Макс. расстояние обнаружения игрока/предмета |
| Угол зрения (FOV) | **100°** | Полный угол конуса зрения |

### Движение

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Скорость патруля | **2.0 wu/с** | Перемещение в Cycle (F-45) |
| Скорость преследования | **4.0 wu/с** (ниже игрока 5.0 — можно оторваться) | Перемещение в Угрозе (F-46) |

### Угроза ([F-42](../03-threat-logic.md), [F-44](./01-model-visual.md))

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Режим доставки | `Contact` | Урон при касании |
| Урон за попадание | **10** | Величина урона за один контакт (в будущую систему здоровья) |
| Порог прекращения преследования | **57.6 wu** (1 сегмент ≈ 1 чанк, 16×16 клеток) | Расстояние бросания погони (F-46) |
| Интервал контакта | **0.7 с** | Мин. время между дозами урона при непрерывном касании |

### Патруль ([F-45](./02-behavior-logic.md))

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Размер сегмента патруля | **16 клеток** (1 чанк ≈ 57.6 wu) | Границы области патруля |

### Сбитие с толку ([F-41](../01-states-logic.md))

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Длительность стана | **2.5 с** | Время неподвижности после попадания мячом (`Stun()`, триггер — будущий) |
| Радиус отвлечения | **1.5 wu** | На каком расстоянии предмет считается «достигнутым» |

> Числовые умолчания проставляются при первой реализации (аналогично `TickSeconds`/`FocusMinDistance` в [REQ-0013](../../REQ-0013-vintage-camera/design.md)); ключевые калибровочные значения — скорость преследования относительно игрока, дальность/FOV и порог прекращения преследования.
