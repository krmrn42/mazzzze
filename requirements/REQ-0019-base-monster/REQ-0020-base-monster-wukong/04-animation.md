## Анимация Wukong

Модель волка — **объёмная 3D-фигура**, а не статичный спрайт: она разворачивается по
направлению движения и проигрывает анимации из своего `.glb`.

| Аспект | Поведение |
|--------|-----------|
| **Ориентация** | Корпус волка развёрнут **носом по направлению движения/взгляда** (совпадает с конусом зрения, [F-40](../02-perception-logic.md)). При смене направления волк поворачивается. |
| **Локомоция** | Пока волк движется (патруль/погоня), проигрывается **зацикленная анимация** (движение корпуса/лап). Темп клипа **ускоряется с ростом скорости** (погоня заметно живее патруля). |
| **Атака** | При контакте с игроком проигрывается **одноразовая анимация укуса/атаки**; по её завершении волк возвращается к локомоции. |
| **Покой** | Отдельного клипа покоя нет — на месте продолжается тот же зацикленный клип на низком темпе. |

### Ограничение ассета

Модель `black_myth_wukong.glb` содержит **только боевые клипы** (атаки/укус) — **готового цикла
ходьбы/бега в ассете нет**. Поэтому в качестве локомоции используется единственный зацикленный
боевой клип модели (даёт непрерывное движение корпуса/лап), а укус — на контакте. Полноценный
цикл ходьбы потребовал бы дополнительной анимации в ассете. Подробности — в [design.md](./design.md).
