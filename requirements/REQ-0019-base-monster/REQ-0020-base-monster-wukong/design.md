## Структура кода

- **`src/Wukong.cs`** — конкретизация `Monster` ([REQ-0019 design.md](../design.md)): в конструкторе
  фиксирует параметры (F-39). Вся механика (восприятие, автомат, движение, урон) — в базе.
  - `TypeId = "wukong"`, `Size = Large`, `Delivery = Contact`.
  - `ModelPath = res://art/black_myth_wukong.glb` (модель + текстуры `black_myth_wukong_*.png`).
- **Модель волка** низкая и длинная; масштаб — по горизонтальному габариту `TargetLength = 2.8` wu
  (масштаб по высоте раздул бы фигуру в разы — см. [REQ-0019 design.md](../design.md) «Модель и масштаб»).
  Коррекция up-оси не нужна (`ModelUprightPitchDeg = 0` — ассет уже Y-up, стоит на ногах).
- Инстанцируется отладочным `MonsterSpawner` рядом со стартом; своей `.tscn` нет.

## Поведение (конкретика типа)

- **Патруль (F-45):** форма `Cycle` из базы = патруль сегмента. Сегмент = чанк, содержащий точку
  спавна (`SegmentCells = 16` клеток ≈ 57.6 wu, [REQ-0003-core/F-06](../../REQ-0003-core/04-chunking-logic.md)):
  границы вычисляются из клетки спавна. BFS-цель — случайная достижимая клетка пола **внутри
  сегмента**; выход за сегмент в патруле/отвлечении заблокирован (`restrictSegment`).
- **Преследование (F-46):** `ChaseSpeed` (4.0) > `PatrolSpeed` (2.0), ниже скорости игрока (5.0) —
  можно оторваться. Прекращение при `dist ≥ ChaseDropDistance = 57.6` wu (1 сегмент) без видимости.
  Погоня **не** ограничена сегментом (в отличие от патруля).
- **Отвлечение (F-41):** в `Cycle` при виде любого `WorldItem` в сегменте и отсутствии видимого
  игрока — идёт к предмету (финальный прямой подход), достигает (`DistractReachRadius`), возвращается.
- **Контактный урон (F-44):** по планарной дистанции касания, `ContactDamage = 10`, не чаще
  `ContactInterval = 0.7` c; сигнал `PlayerHit` + красная вспышка (система здоровья — будущая).

## Что НЕ входит (границы Wukong)

- **Дальняя атака** отсутствует (строго контактный).
- **Убийство мячом** — Wukong большого размера, мяч его только путает (`Stun()` — хук без триггера,
  [IDEA-0025](../../../ideas/items/IDEA-0025-tennis-ball.md)).
- **Полноценный спавн** — за будущим спавнером; здесь отладочное размещение у старта.
- **Анимации модели** (`.glb`) не проигрываются — при доработке.

## Открытые вопросы (решения)

- **Стриминг:** монстр персистентен (живёт под `Main`, как `WorldItem`), не привязан к чанку — не
  респаунится и не исчезает при выгрузке.
- **Маршрут патруля:** случайная выборка достижимых клеток сегмента (не фиксированные точки).
- **Отвлечение за пределы сегмента:** нет — ограничено сегментом (`restrictSegment`).
