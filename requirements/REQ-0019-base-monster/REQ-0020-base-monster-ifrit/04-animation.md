## Анимация Ifrit

Модель ифрита — **объёмная 3D-фигура**, а не статичный спрайт: она разворачивается по
направлению движения и проигрывает анимации из своего `.glb` (полный набор в ассете).

| Аспект | Поведение |
|--------|-----------|
| **Ориентация** | Ифрит развёрнут **лицом по направлению движения/взгляда** (совпадает с конусом зрения, [F-40](../02-perception-logic.md)). При смене направления поворачивается. |
| **Покой** | На месте (нейтральный цикл без движения) проигрывается зацикленная анимация покоя (idle). |
| **Бег / ходьба** | При движении (патруль/погоня) — зацикленная анимация бега; темп **ускоряется с ростом скорости** (погоня заметно быстрее патруля). |
| **Атака** | При контакте с игроком — одноразовая анимация атаки; по завершении возврат к покою/бегу. |
| **Реакция на попадание** | При стане (попадание предметом) — одноразовая анимация «получил удар» (be-hit). |

Ассет `ifrit.glb` содержит полный набор анимаций (покой, бег, атака, реакция на удар и др.), поэтому
локомоция и боевые реакции читаются естественно. Клип смерти в ассете есть, но не задействован —
состояния смерти пока нет. Имена клипов и детали контроллера — в [design.md](./design.md).
