## F-44: Ifrit — визуальная модель и контактный режим

Ifrit специализировает базовый шаблон ([REQ-0019](../README.md)): конкретная модель, размер и выбор режима доставки урона.

### Тип монстра

| Параметр | Значение |
|----------|----------|
| Идентификатор типа | `ifrit` |
| Тип существа | **Ифрит** (огненный демон-гуманоид) |
| Размер (вариант) | **Большой/обычный** (база) — мячом сбивается с толку, но не убивается ([IDEA-0025](../../../ideas/items/IDEA-0025-tennis-ball.md)) |
| Режим доставки урона | **Контактный** (из двух режимов шаблона [F-42](../03-threat-logic.md)) |
| 3D-модель | `art/ifrit.glb` (файл присутствует в репозитории) |

### Визуальное представление (уточнение [05-visual.md](../05-visual.md))

| Свойство | Значение |
|----------|----------|
| Модель | `art/ifrit.glb` — ифрит |
| Масштаб | Целевая высота — из [параметров по умолчанию](./03-params-data.md); читается как крупная фигура в масштабе коридора |
| Ориентация | Разворачивается по направлению движения/взгляда (конус зрения смотрит вперёд, [F-40](../02-perception-logic.md)) |
| Коллизия | Тело ифрита регистрирует касание игрока (для контактного урона) и попадание брошенного мяча; непроходимо для игрока |

### Контактный режим урона

- Урон игроку наносится **при непосредственном касании** телом Ifrit'а.
- Одно касание снимает одну дозу урона; повторный урон — не чаще, чем раз в `интервал контакта` ([параметр](./03-params-data.md)).
- Размер урона за попадание — из [параметров по умолчанию](./03-params-data.md); применяется к будущей системе здоровья игрока.

> Дальняя атаки у Ifrit **нет**: режим доставки строго контактный. Атака на расстоянии — за будущим дальним типом монстра.
