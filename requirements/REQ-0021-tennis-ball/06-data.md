# Теннисный мячик — Параметры по умолчанию (Data)

> **Фича:** REQ-0021-tennis-ball · **Тип документа:** требования, грань «данные» (WHAT, без деталей реализации)

Конкретные значения параметров мячика. Числа — по умолчанию; калибруются при реализации (аналогично `TickSeconds`/`FocusMinDistance` в [REQ-0013](../REQ-0013-vintage-camera/design.md)).

## Идентичность и модель

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Идентификатор типа | `tennis_ball` | Ключ типа предмета |
| Отображаемое имя | «Теннисный мячик» | UI/инвентарь |
| Паттерн использования | B (Activated) | Активация «в руку» |
| Расходуемость | нет (многоразовый) | Бросок не уничтожает |
| Путь к 3D-модели | `art/base_ball.glb` | Модель в мире/руке/снаряде |
| Размер в мире | (доля от роста игрока; ≈ как у прочих предметов) | Масштаб модели `WorldItem` |

## Зарядка и бросок ([F-48](./02-throw-logic.md), [F-50](./04-input.md))

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| MinThrowForce | (калибруется) | Сила броска при тапе / минимуме заряда |
| MaxThrowForce | (калибруется) | Сила при полном заряде |
| MaxChargeTime | (калибруется; ≈ 1 с) | Время выхода на максимальную силу; далее плато |
| UpwardBias | (калибруется) | Вертикальная составляющая начальной скорости (формирует дугу) |
| MoveSpeedFactorWhileCharging | (калибруется; < 1) | Множитель скорости ходьбы во время заряда |

## Физика снаряда ([F-48](./02-throw-logic.md))

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| Гравитация | (стандартная сцены) | Ускорение вниз |
| BounceDamping | (калибруется) | Потеря энергии при отскоке от стен/пола |
| StopThreshold | (калибруется) | Порог скорости «считается остановившимся» |
| StopTime | (калибруется) | Сколько времени скорость ниже порога → переход в `WorldItem` |
| MaxAirborneTime | (предохранитель) | Принудительная остановка при зависании (бесконечный отскок) |

## Воздействие на монстра ([F-49](./03-monster-effect-logic.md))

| Параметр | Умолчание | Смысл |
|----------|-----------|-------|
| StunDuration | (калибруется; согласуется с `длительностью стана` монстра, [REQ-0019/F-39](../REQ-0019-base-monster/04-data.md)) | Длительность стана большого монстра при прямом попадании |
| InspectDuration | (калибруется) | Сколько монстр «осматривает» достигнутый мяч перед возвратом к циклу |
| InspectRadius | (калибруется) | Расстояние до мяча, считаемое «достигнуто» при отвлечении |

> Длительность стана и радиус отвлечения дублируют/согласуются с параметрами шаблона монстра ([REQ-0019/F-39](../REQ-0019-base-monster/04-data.md)); мяч лишь инициирует переход в `Disrupted`, длительность живёт в монстре.
