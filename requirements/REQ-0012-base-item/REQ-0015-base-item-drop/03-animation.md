## F-28: Анимация выброса

При выбросе предмет визуально «вылетает» из игрока яркой точкой и приземляется, превращаясь в модель.

| Элемент | Требование |
|---------|-----------|
| Снаряд | Яркая светящаяся точка, воспринимается как маленькая «звёздочка» (мерцающая) |
| Траектория | Небольшая парабола: от игрока (на уровне корпуса) вверх-вперёд и вниз к точке приземления |
| Дальность | Приземление на расстоянии из F-26 (по умолчанию 3 корпуса ≈ 1.8 м) перед игроком |
| Длительность полёта | Короткая, конфигурируемо (по умолчанию ≈ 0.6 с) |
| Высота дуги | Конфигурируемо (по умолчанию ≈ 1.2 м над прямой) |
| Завершение | В точке приземления точка исчезает, и появляется 3D-модель предмета — переход к [REQ-0014](../REQ-0014-base-item-item-in-world/README.md) (анимация появления модели описана там, F-25) |

**Ощущение:** точка яркая настолько, чтобы «зацвести» в свечении (bloom сцены уже включён), с лёгким мерцанием/пульсацией размера — «звёздочка», а не просто шар.

## Конфигурируемые параметры

| Параметр | По умолчанию | Смысл |
|----------|--------------|-------|
| `DropFlightDuration` | 0.6 с | Время полёта звёздочки |
| `DropArcHeight` | 1.2 м | Высота параболы над прямой старт→цель |
