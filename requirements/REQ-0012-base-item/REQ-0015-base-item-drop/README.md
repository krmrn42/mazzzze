## US-15: Выбрасывание предмета из рюкзака

**Как** игрок,
**я хочу** выбрасывать предметы из рюкзака обратно в лабиринт,
**чтобы** освобождать слоты и оставлять предметы в мире.

**Критерии приёмки:**
- Выброс доступен для **любого** предмета из инвентаря и работает одинаково для всех типов.
- Управление (два равнозначных способа, см. F-27):
  - **Shift + цифра-колонки** после выбора ряда — мгновенный выброс ячейки;
  - **курсор + G** — выброс предмета под текущим курсором ячейки.
- После выброса предмет исчезает из слота, слот освобождается.
- Проигрывается анимация: яркая точка (маленькая «звёздочка») летит по небольшой параболе (F-28).
- Точка приземляется **перед игроком** на расстоянии 3 корпусов (диаметров) игрока — конфигурируемо.
- Если впереди стена ближе цели — предмет приземляется, не доходя до стены.
- В точке приземления точка превращается в соответствующую 3D-модель предмета (переход в состояние InWorld, см. [REQ-0014](../REQ-0014-base-item-item-in-world/README.md)).
