# Предмет в мире — Визуальное представление (Visual)

> **Фича:** REQ-0014-base-item-item-in-world · **Functional:** F-24
> **Тип документа:** требования, грань «визуал» (WHAT, без деталей реализации)

---

## F-24: Представление предмета в мире

| Аспект | Требование |
|--------|-----------|
| Модель | 3D-модель типа предмета (`.glb`), та же, что используется для иконки инвентаря |
| Размер | Уменьшена до `WorldItemSizeFraction × PlayerHeight` (по умолчанию 1/8 × 1.8 м ≈ 0.22 м по высоте) |
| Единица масштаба | Масштабирование по **высоте** модели (её вертикальному габариту), пропорционально по всем осям |
| Положение | Стоит на полу: нижняя грань модели касается уровня пола в точке размещения |
| Точка | Мировая позиция (задаётся создателем — например, точкой приземления выброса, [REQ-0015](../REQ-0015-base-item-drop/README.md)) |
| Ориентация | Произвольная/нейтральная (специальный разворот не требуется) |
| Время жизни | Постоянно, пока не подобран; не зависит от загрузки/выгрузки чанков |

## Конфигурируемые параметры

| Параметр | По умолчанию | Смысл |
|----------|--------------|-------|
| `WorldItemSizeFraction` | 0.125 (1/8) | Доля роста игрока — высота модели в мире |
| `PlayerHeight` | 1.8 м | Эталон роста игрока |

**Границы:** коллизия/физика предмета в мире и подбор (`InWorld → InInventory`) — вне этого требования (задел на будущее).
