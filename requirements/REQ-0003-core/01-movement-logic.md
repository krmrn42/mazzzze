# Ядро — Перемещение и персонаж (Movement)

---

## US-01: Перемещение по лабиринту

**Как** игрок,
**я хочу** перемещаться по коридорам лабиринта с помощью клавиш W, A, S, D,
**чтобы** исследовать мир в поисках выхода.

**Критерии приёмки:**
- При нажатии W игрок движется вперёд (туда, куда смотрит камера)
- При нажатии S игрок движется назад
- При нажатии A игрок движется влево
- При нажатии D игрок движется вправо
- Скорость движения постоянна и комфортна для навигации по коридорам
- При отпускании всех клавиш игрок останавливается
- Игрок не может проходить сквозь стены
- Игрок стоит на полу (не проваливается, не летает)
- Движение возможно только внутри коридоров; стены — непреодолимое препятствие

---

## F-03: Персонаж игрока

**Физические характеристики:**
- Персонаж имеет коллизию: не может проходить сквозь стены
- Персонаж подвержен гравитации: падает вниз при отсутствии опоры
- Размер персонажа (коллизия): существенно меньше ширины коридора, чтобы игрок мог комфортно маневрировать
- Модель персонажа видна перед камерой
- Модель ориентируется лицом в направлении движения
