## F-05: Управление

| Действие | Устройство | Клавиша/элемент |
|----------|-----------|-----------------|
| Движение вперёд | Клавиатура | W |
| Движение назад | Клавиатура | S |
| Движение влево | Клавиатура | A |
| Движение вправо | Клавиатура | D |
| Осмотр по горизонтали | Мышь | Движение влево/вправо |
| Осмотр по вертикали | Мышь | Движение вперёд/назад |
| Приблизить камеру | Мышь | Колёсико «на себя» |
| Отдалить камеру | Мышь | Колёсико «от себя» |

Направление движения всегда относительно того, куда смотрит камера:
- W = движение туда, куда направлен взгляд (в плоскости пола)
- S = движение в противоположную сторону
- A = движение влево от направления взгляда
- D = движение вправо от направления взгляда
