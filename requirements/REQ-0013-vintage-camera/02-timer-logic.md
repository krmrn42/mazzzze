## F-22: Таймер обратного отсчёта видоискателя

Таймер запускается одновременно с открытием видоискателя и управляет моментом фактической съёмки (создания фотографии). Таймер работает только при соблюдении фокусного расстояния (см. [F-23](./README.md#f-23-фокусное-расстояние-focus-distance)).

### Параметры таймера

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| Стартовое число | 3 | С какого числа начинается обратный отсчёт |
| Длительность тика | ~0.67 секунды | Интервал между сменой цифр (отсчёт ускорен ×3) |
| Всего тиков | 3 | Количество тиков (3 → 2 → 1) |
| Отображение 0 | Не отображается | После 1 сразу происходит срабатывание без показа 0 |

Длительность тика является **настраиваемым параметром** (в коде, через конфиг или экспортируемую переменную). Значение по умолчанию — ~0.67 секунды (весь отсчёт ускорен в 3 раза).

### Отображение цифр

| Параметр | Значение |
|----------|----------|
| Позиция | Строго по центру окна видоискателя (и по горизонтали, и по вертикали) |
| Размер | Крупный, перекрывает значительную часть видоискателя, но не загораживает его полностью (примерно 30–50% высоты окна) |
| Прозрачность | Полупрозрачная — сквозь цифру видно изображение видоискателя |
| Шрифт | Крупный, жирный, стилизованный под старину (с засечками / serif) |
| Цвет | Светлый (белый или кремовый) — читается поверх любого фона видоискателя |
| Анимация смены | Смена цифр может сопровождаться короткой анимацией (например, мерцание или плавное появление/исчезновение). Конкретный стиль анимации уточняется на этапе реализации. |

### Алгоритм

1. **t = открытие видоискателя.** Начинается первый интервал. Отображается цифра **3**.
2. **t + одна длительность.** Отображается цифра **2** (первый тик завершён).
3. **t + две длительности.** Отображается цифра **1** (второй тик завершён).
4. **t + три длительности.** Цифра 1 сменяется — но 0 не отображается. Вместо этого мгновенно происходит:
   - Создание предмета «Фотография» ([REQ-0017-photo](../REQ-0017-photo/))
   - Фотография помещается в освободившийся слот
   - **Жест взятия предмета в руку + подсветка ячейки** — чтобы игрок заметил новый предмет
   - Закрытие видоискателя
   - Уничтожение фотоаппарата

Цифра **0 никогда не отображается** — по достижении конца третьего интервала сразу срабатывает финальное действие.

### Прерывание таймера

Таймер может быть прерван до завершения в следующих случаях:

| Причина | Поведение |
|---------|-----------|
| Нарушение фокусного расстояния | Таймер сбрасывается, видоискатель закрывается. Фотоаппарат возвращается в состояние Activated (НЕ уничтожается) |
| Выбрасывание (drop activated) | Таймер останавливается, видоискатель закрывается. Фотоаппарат выбрасывается в мир |
| Деактивация (deactivate) | Таймер останавливается, видоискатель закрывается. Фотоаппарат возвращается в забронированный слот |

Во всех случаях прерывания фотография **не создаётся**, ресурс фотоаппарата **не расходуется**.

### Визуальная схема таймлайна

```
Открытие     Тик 1        Тик 2        Срабатывание
   |           |            |               |
   v           v            v               v
  [3] ------> [2] ------> [1] ----------> фото + слот + уничтожение
   ^           ^            ^               ^
   |           |            |               |
   +-- 0.67с --+-- 0.67с --+-- 0.67с ------+

В любой момент может быть прервано:
   - нарушение фокусного расстояния → сброс, камера остаётся
   - drop activated / deactivate → сброс, стандартный переход
```

Полная длительность от открытия видоискателя до срабатывания: 3 × тик = ~2.0 секунды (по умолчанию, отсчёт ускорен ×3).
