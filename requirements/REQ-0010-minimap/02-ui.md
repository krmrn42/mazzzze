## F-09: Мини-карта — визуальный стиль

**Тип:** стилизация под нарисованную от руки карту на старом пергаменте, постепенно проявляющуюся по мере исследования.

**Элементы отрисовки:**

| Элемент | Описание |
|---------|----------|
| Фон | Текстура старого пергамента (тёплый бежевый тон, неравномерный) |
| Пол (исследованная клетка) | Светлое пятно на пергаменте — как будто проступило |
| Стена (исследованная клетка) | Тёмное пятно/штриховка — непроходимая область |
| Неисследованная область | Скрыта туманом войны (пергамент затемнён или отсутствует) |
| Игрок | Чёткий значок (треугольник или стрелка), указывающий направление взгляда |
| Вход/выход (после посещения) | Особый значок (арка/проём) на границе исследованной области |
| Ближняя зона (полная детализация) | Клеточная сетка пол/стена, каждый квадрат различим |
| Дальняя зона (схематично) | Сплошной контур исследованной территории без разделения на клетки |

**Размер виджета:** классический — примерно 15–20% ширины экрана.
