# Мини-карта — Управление (Input)

> **Фича:** REQ-0010-minimap · **Functional:** F-11
> **Тип документа:** требования, грань «ввод» (WHAT, без деталей реализации)

---

## F-11: Мини-карта — управление

| Действие | Устройство | Клавиша/элемент |
|----------|-----------|-----------------|
| Приблизить миникарту | Мышь | Ctrl + колёсико «на себя» |
| Отдалить миникарту | Мышь | Ctrl + колёсико «от себя» |
| Переключить ориентацию | Клавиатура | Tab |

**Режимы ориентации:**
- **По умолчанию:** миникарта вращается синхронно с камерой (направление вперёд — вверх)
- **Альтернативный:** север фиксирован сверху
- Переключение между режимами — по Tab
- Сохранение состояния между сессиями — см. отдельный документ по сохранению/восстановлению игры
