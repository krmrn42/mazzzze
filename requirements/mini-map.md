# Maze Explorer — Mini-Map

> **Document type:** Feature requirements.
> **Audience:** Game designers, product owners, AI agents.
> **No implementation details** — this document describes WHAT the mini-map does, not HOW.

## US-10: Мини-карта

**Как** игрок,
**я хочу** видеть миниатюрную схему лабиринта с моим текущим положением,
**чтобы** лучше ориентироваться в исследованном пространстве, не имея полноценной карты.

**Критерии приёмки:**
- Мини-карта всегда видна в левом верхнем углу экрана
- На миникарте видна ближняя зона (N×N клеток вокруг игрока) с полной детализацией: каждая клетка — пол или стена
- За пределами ближней зоны исследованная территория показана схематично — общим контуром, без деталей коридоров
- Неисследованные области скрыты туманом войны
- Отображается текущая позиция игрока и направление его взгляда
- Мини-карта по умолчанию вращается вместе с камерой игрока (вперёд всегда вверх)
- Игрок может переключить ориентацию на «север сверху» клавишей Tab
- Вход и выход отображаются на миникарте только после того, как игрок их посетил (вошёл в клетку входа или вышел в клетку выхода)
- Игрок может приближать и отдалять миникарту с помощью Ctrl + колёсико мыши
- Приближение ограничено минимальным пределом (нельзя упереться в одну клетку)
- Отдаление ограничено максимальным пределом (нельзя увидеть весь лабиринт целиком)

## F-09: Мини-карта — визуальный стиль

**Тип:** стилизация под нарисованную от руки карту на старом пергаменте, постепенно проявляющуюся по мере исследования.

**Элементы отрисовки:**

| Элемент | Описание |
|---------|----------|
| Фон | Текстура старого пергамента (тёплый бежевый тон, неравномерный) |
| Пол (исследованная клетка) | Светлое пятно на пергаменте — как будто проступило |
| Стена (исследованная клетка) | Тёмное пятно/штриховка — непроходимая область |
| Неисследованная область | Скрыта туманом войны (пергамент затемнён или отсутствует) |
| Игрок | Чёткий значок (треугольник или стрелка), указывающий направление взгляда |
| Вход/выход (после посещения) | Особый значок (арка/проём) на границе исследованной области |
| Ближняя зона (полная детализация) | Клеточная сетка пол/стена, каждый квадрат различим |
| Дальняя зона (схематично) | Сплошной контур исследованной территории без разделения на клетки |

**Размер виджета:** классический — примерно 15–20% ширины экрана.

## F-10: Мини-карта — память и состояние

Мини-карта отслеживает историю перемещений игрока для отображения тумана войны.

**Требования:**
- Хранятся последние 1000 посещённых клеток (FIFO-буфер)
- При заполнении буфера старые клетки забываются — туман смыкается позади
- Посещение клетки фиксируется, когда игрок входит в неё
- Вход и выход сохраняются вне буфера (не забываются после вытеснения из FIFO)
- Состояние мини-карты сбрасывается при новом запуске игры

## F-11: Мини-карта — управление

| Действие | Устройство | Клавиша/элемент |
|----------|-----------|-----------------|
| Приблизить миникарту | Мышь | Ctrl + колёсико «на себя» |
| Отдалить миникарту | Мышь | Ctrl + колёсико «от себя» |
| Переключить ориентацию | Клавиатура | Tab |

**Режимы ориентации:**
- **По умолчанию:** миникарта вращается синхронно с камерой (направление вперёд — вверх)
- **Альтернативный:** север фиксирован сверху
- Переключение между режимами — по Tab
- Сохранение состояния между сессиями — см. отдельный документ по сохранению/восстановлению игры
