# IDEA-0014 — Биомы лабиринта

> **Область:** world · **Статус:** 🟡 идея

Районы с разной палитрой/материалом стен (песок, тёмный камень, лёд/корни) — визуальное разнообразие и «зоны» для запоминания. Генератор уже детерминирован ([MazeData](../../requirements/TECH_SPEC.md#51-mazedata---world-data-and-procedural-generation)), биом можно выводить из хэша региона.
